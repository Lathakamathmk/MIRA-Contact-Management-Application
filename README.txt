Developed By:
Latha Kamath M K

Project Overview:
This project is a Contact Management Application developed for the Junior Python Fullstack Developer evaluation task.

The application allows users to:
- Create contacts
- View contacts
- Update contacts
- Delete contacts
- Search contacts

Technology Stack:
- Python Flask
- SQLite
- HTML
- CSS
- JavaScript
- Flask-SQLAlchemy
- Flask-CORS

Features:
- CRUD Operations
- Search Functionality
- Email Validation
- Phone Number Validation
- Duplicate Email Prevention
- Persistent Database Storage

Project Structure:
- app.py → Backend Flask application
- templates/index.html → Frontend UI
- requirements.txt → Python dependencies
- contacts.db → SQLite database

How to Run the Project:

1. Install Python

2. Install required packages:
pip install -r requirements.txt

3. Run the application:
python app.py

4. Open browser and visit:
http://127.0.0.1:5000
